# C1 Quick Ops — Szybkie komendy

## Codzienne

| Powiedz Claude                  | Co zrobi                                          |
|---------------------------------|--------------------------------------------------|
| `codzienny przegląd`            | Morning review: Gmail + Calendar + Pipeline      |
| `sprawdź maile`                 | Skanuje Gmail, klasyfikuje odpowiedzi, update CRM|
| `szukaj leadów [niche]`         | Apollo search → enrich → score → Notion CRM      |
| `pokaż pipeline`               | Status leadów z Notion Pipeline (per Etap)       |
| `wyślij outreach`              | Cold emails → drafty w Gmail → Outreach db       |
| `follow-upy`                   | Follow-upy do leadów Contacted > 3 dni           |
| `raport`                       | KPI: leady/maile/odpowiedzi/spotkania            |

## Spotkania

| Powiedz Claude                  | Co zrobi                                          |
|---------------------------------|--------------------------------------------------|
| `zaplanuj spotkanie z [firma]`  | Calendar slot + event + potwierdzenie Gmail       |
| `przygotuj się na spotkanie`    | Bigdata research + CRM history + brief            |
| `notatki ze spotkania [firma]`  | Notion notatki + Pipeline update + follow-up      |

## Content & Sales
